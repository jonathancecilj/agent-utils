---
description: Research a topic using web search and summarize findings.
---

1. Ask the user for the **Research Topic**.
2. Ask if they want to focus on **News**, **Policy**, or **General Technical Info**.
3. Perform a web search (`search_web`) for the topic.
    - If "News", append "latest news" or "recent developments" to the query.
    - If "Policy", append "policy regulations compliance" to the query.
4. Read the content of the top 3 relevant results.
5. Create a summary file in `research-notes/{{date}}-{{topic-slug}}.md`.
6. The summary must include:
    - **Key Findings**
    - **Relevant Dates/Events**
    - **Links/Sources**
    - **Raw Notes** for further drafting.
7. Open the summary file.
